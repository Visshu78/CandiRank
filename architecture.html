Track 01 — Data & AI Challenge

Intelligent Candidate Discovery

Complete system architecture — original core pipeline plus every differentiator feature layered in, end to end from raw data to a ranked, explainable shortlist.

00
Data Ingestion
Candidate dataset
Profiles, skills, experience, education, activity logs — CSV/JSON
Job descriptions
Raw JD text or structured JSON input
01
Preprocessing & Deep Understanding
Profile cleaning & normalization
Deduplication, skill taxonomy mapping ("JS" = "JavaScript")
LLM-based JD parser
Extracts required skills, nice-to-have, seniority, domain, intent into structured JSON
Hybrid keyword pass new
BM25 keyword layer runs alongside parsing to catch exact must-have terms semantic models might miss
02
Embedding & Semantic Index
Candidate vector embeddings
sentence-transformers encodes every profile into the same vector space
JD vector embedding
Job description encoded into the matching embedding space
FAISS / ChromaDB index
Approximate nearest-neighbor search — fast top-K retrieval at scale
Caching layer new
Precomputed embeddings cached to disk — repeated queries skip recomputation entirely
03
Multi-Signal Scoring Engine
Semantic score
Cosine similarity between JD and candidate vectors
Profile score
Skills, experience, education match against parsed JD requirements
Behavioral score
Recency of activity, platform engagement, career growth trajectory
Dynamic weight tuning new
Weights shift by role type — senior roles favor profile/experience, fresher roles favor behavioral activity
Cross-encoder re-ranking new
Top-50 candidates re-scored with a precision cross-encoder for a final accuracy boost
Cold-start fallback new
Candidates with sparse activity data get a fallback score so they aren't unfairly buried
04
Ranking & Trust Layer
Ranking engine
Sorts by final weighted score, returns top-N shortlist
Explainability module new
Plain-language reasoning per candidate: "91% semantic match · active 2 days ago · positive career growth"
Bias / fairness audit new
Checks score distribution across groups to surface and reduce unintended bias
05
Output Layer
Ranked shortlist export
CSV / JSON in the predefined submission format — rank, score, match reason
GitHub repo + README
Organized code, clear documentation of methodology and architecture
06
Bonus — Demo & Presentation Layer
Streamlit live demo new
Judges paste a JD and watch the ranked shortlist generate in real time
Speed benchmark display new
Quantified performance shown live — "10,000 candidates ranked in 2.3 seconds"
